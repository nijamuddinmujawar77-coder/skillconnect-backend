## üìù How to Take Screenshots

### Using Windows:
- **Full Screen:** Press `Windows + Print Screen`
- **Selected Area:** Press `Windows + Shift + S`
- **Active Window:** Press `Alt + Print Screen`

### Using Browser Dev Tools:
1. Press `F12` to open dev tools
2. Click device toolbar icon (mobile icon)
3. Select device (iPhone X, iPad, etc.)
4. Take screenshot

### Professional Screenshots Tips:
- Use consistent window size
- Clear browser cache before screenshots
- Close unnecessary tabs
- Use incognito mode for clean screenshots
- Ensure good lighting/contrast
